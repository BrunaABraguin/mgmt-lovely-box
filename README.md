# MGMT Lovely Box

Gerenciamento e-commerce de Produtos e Clientes da loja Lovely Box.
[Live Application](https://mgmt-lovely-box.netlify.app/)

## Instalação

```bash
npm init
```
## Utilização

1. Inicie a aplicação REACT:
```bash
npm start
```

2. Inicie o servidor JSON Server:
```bash
npm run server
```
